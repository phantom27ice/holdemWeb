# IMPLEMENTATION_NOTES.md

> 本文档用于交付前说明：规则实现边界、已知限制、后续扩展位。

## 1. 规则实现对齐（基于 `TEXAS_HOLDEM_RULEBOOK.md`）

1. 已实现：德州 6-max 单桌、无上限注（No-Limit）核心流程。
2. 已实现：最小加注与 short all-in reopen 累计规则。
3. 已实现：主池/边池分层、逆序结算、平分与 odd chip 分配。
4. 已实现：未跟注金额返还（uncalled return）。
5. 已实现：摊牌 7 选 5 评估（10 类牌型，含 wheel straight）。

## 2. 当前已知限制

1. 当前仅支持本地单机牌桌，不含联机、断线重连、房间系统。
2. AI 为规则驱动的被动策略（优先 check/call），非策略型对手。
3. UI 动画为轻量事件动画，未做复杂时间轴与可配置特效系统。
4. 暂无完整资金系统（买入、结算大厅、历史战绩持久化）。

## 3. 非目标范围（当前版本）

1. 锦标赛盲注级别与升盲时间轴。
2. 保险、Run It Twice、Straddle 等扩展房规。
3. 聊天、观战、战绩排行、成就系统。

## 4. 后续扩展位（建议顺序）

1. P1：加入更真实 AI 决策层（位置、底池赔率、牌力估计）。
2. P2：引入局内操作日志面板（动作历史、底池变化、结算明细）。
3. P3：把桌面状态抽离为可回放记录（便于复盘与争议核对）。
4. P4：联机协议与服务端裁决（前端仅展示，服务端做规则真值）。
